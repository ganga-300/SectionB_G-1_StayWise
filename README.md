# StayWise: Hotel Booking Analytics for Revenue & Cancellation Optimization

## 📌 Project Overview
This project analyzes hotel booking data to identify factors affecting cancellations, customer behavior, and revenue generation. The goal is to provide actionable insights to improve occupancy rates and optimize pricing strategies.

## 🎯 Problem Statement
Analyze hotel booking data to identify the key factors influencing booking cancellations, customer behavior, and revenue generation, and recommend strategies to improve occupancy and reduce cancellations.

## 📊 Dataset
- Source: Kaggle - Hotel Booking Dataset
- Contains 119,000+ records with booking details, customer data, and pricing information.

## 🛠 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Tableau Public
- GitHub

## 📈 Key Analysis Areas
- Booking cancellation patterns
- Customer segmentation
- Revenue trends
- Seasonal demand analysis
